# Практическая работа 4 - Поддержка и тестирование программных модулей - Оп. Н.

## Цель работы
Приобрести практические навыки ручного тестирования методом "белого ящика".

## Разработчик
Оп. Н.

## Формулы
Из варианта 13.

## Стек
- C# / WPF (.NET 10)
- OxyPlot
- Visual Studio 2026

## Архитектура
- `MainWindow` - Точка входа
- `Pages/MainPage` - Главная с кнопками заданий
- `Pages/Task1Page, Pages/Task2Page, Pages/Task3Page` - Страницы заданий
- `Controls/BackButton` - Кнопка возврата на главную для страниц заданий
- `images/` - изображения формул (1.jpg, 2.jpg, 3.jpg)

## Запуск
1. Открыть решение в Visual Studio
3. Положить `1.jpg`, `2.jpg`, `3.jpg` в `bin/Debug/???/images/` (знаки вопроса означают название директори, к примеру это у меня net10.0-windows)
4. F5
